# 💰 FinTrack Pro

A modern and responsive Personal Finance Tracker built using **HTML, CSS, and Vanilla JavaScript**. FinTrack Pro helps users manage their income and expenses with an intuitive dashboard, transaction management, analytics, and local data persistence.

---

## 🚀 Features

### 🔐 Authentication

* User Registration
* User Login
* Logout Functionality
* Local Authentication using LocalStorage

### 📊 Dashboard

* Current Balance
* Total Income
* Total Expense
* Total Transactions
* Responsive Financial Overview Cards

### 💸 Transaction Management

* Add New Transaction
* Edit Existing Transaction
* Delete Transaction
* Search Transactions
* Filter by:

  * All
  * Income
  * Expense
* Automatic Statistics Update

### 📈 Analytics

* Cash Flow Chart using Chart.js
* Income vs Expense Progress Bars
* Dynamic Analytics Updates

### ⚙️ Settings

* Dark / Light Theme
* Update Profile Name
* Currency Selection
* Reset All Application Data

### 💾 Local Storage

* User Authentication Data
* Transactions
* Theme Preference
* Selected Currency
* Profile Information

### 📱 Responsive Design

* Desktop Layout
* Tablet Support
* Mobile Friendly Navigation
* Modern UI Components
